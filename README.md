# DBSCAN-and-HDBSCAN-for-Gaia-DR3-clustering-detection

This is a project developed as part of the final project of my Master's Degree: Artificial Intelligence.
Here, data downloaded from Gaia DR3 is studied with non-supervised clustering alogrithms (DBSCAN & HDBSCAN) to find star clusters.
In this repository you can find the suource code of the project both in .ipynb and .py format.
